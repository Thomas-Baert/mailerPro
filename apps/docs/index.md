# MailerPro Documentation

Bienvenue dans la documentation technique de **MailerPro**, la solution d'e-mailing intelligente.

::: tip PROJET MONOREPO
Ce projet utilise TurboRepo pour gérer le Frontend (React), le Backend (Fastify) et cette Documentation (VitePress) dans un seul dépôt.
:::

## Structure du Projet

- **Apps/Web** : Interface utilisateur React 19 + Vite.
- **Apps/API** : Backend Fastify + Prisma.
- **Packages/Database** : Schéma Prisma et client database partagé.
